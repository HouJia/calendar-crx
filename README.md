# calendar.crx
谷歌插件，日历，可查看农历、可设置大小周

* 引入该插件：

  1. 下载源码

  2. 谷歌浏览器 -> 扩展程序 -> 开发者模式 -> 加载已解压的扩展程序
* Google商店地址：https://chromewebstore.google.com/detail/%E8%8A%82%E5%81%87%E6%97%A5%E6%97%A5%E5%8E%86/nbnocjjjcabknnadklnobhnlnlbgcfnj
